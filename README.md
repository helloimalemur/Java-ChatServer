# Java-ChatServer
Simple multi-threaded chat server using sockets

<h3>commands;</h3>
<b>/help</b> , <b>/nickname</b> to change nickname, <b>/quit</b> to exit

<h3>Linux:</h3>

Server: ```./run.sh```

Client: ```./run.sh client```

<h3>Windows:</h3>

Server: ```./run.ps1```

Client: ```./run.ps1 client```
